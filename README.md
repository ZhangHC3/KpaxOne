# KpaxOne｜K星一号
Android cross‑device remote control project for Windows PC.

## Overview
This project enables your Android phone to remotely control a Windows computer over the local network.
It covers basic device control, audio relay, call management, and will integrate Claude Code for advanced AI‑powered automation in the future.

## Core Features
- Power on / shutdown Windows PC remotely
- Adjust system volume, control media playback
- Stream phone microphone audio to PC in real‑time
- Toggle WeChat & Doubao voice calls
- Low‑latency local network communication

## Long‑term Plan
Integrate Claude Code to implement intelligent task scheduling, complex automation and natural‑language‑driven PC control.

# KpaxOne｜K星一号
安卓跨设备远程控制 Windows 电脑项目。

## 项目简介
本项目实现安卓手机在局域网内远程操控 Windows 电脑，
包含设备基础控制、音频传输、通话管理，并计划接入 Claude Code，实现 AI 驱动的高级自动化能力。

## 核心功能
- 远程控制电脑开关机
- 系统音量调节、音乐播放控制
- 手机麦克风实时音频推流至电脑
- 微信、豆包通话开启/关闭控制
- 低延迟局域网通信

## 长期规划
接入 Claude Code，实现智能任务调度、复杂自动化、自然语言控制电脑等高级AI功能。
